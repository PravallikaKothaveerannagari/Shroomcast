About

My father-in-law to be is a passionate mushroom picker. This app is based on the insights he gathers before going out to pick various types of mushrooms.

I hope it will prove useful to you too.

How to use the app

To use Shroomcast you will need Visual Crossing account (Shroomcast uses this weather API to gather information). If you don't have an account yet, you can create one here:
https://www.visualcrossing.com/

Sections of the app

The Shroomcast contains of 4 sections:

  1. authorization
  2. forecast
  3. query count check
  4. about

I will describe some them in more detail in the following sections

Authorization

In this section you put yout Visual Crossing authorization key. You can find it in your Visual Crossing Account:
https://www.visualcrossing.com/account

Forecast

The main view of this app. After providing valid Visual Crossing key in authorization section, you can put the municipality in the forecast view and choose the date to get weather data useful to predict if you find mushrooms in the areas they could grow.

Query

Visual Crossing has several pricing plans. One of them is free, but in it you can spend 50 query points and after that you won't be able to send any more queries during this day. This section gathers the data about the query you spent using the app during the day for you to see how many query points you have left.

And that's all

Happy weather shroomcasting!
